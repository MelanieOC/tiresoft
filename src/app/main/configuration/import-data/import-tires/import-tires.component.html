Pasos para importar los neumáticos

PASO 1 Solo arrastre el archivo excel de la plantilla realizada

PASO 2 Una vez seleccionado el archivo el sistema verificará si es correcto el archivo

PASO 3 Para verificar si los datos se relacionaron con el sistema, haz click en Ir al registro de Neumáticos y te rediccionará a la lista de Neumáticos.

report_problem Antes de importar su excel verificar que los datos no se repliquen en las columnas 'disenio por reencauche' y 'modelo'.
