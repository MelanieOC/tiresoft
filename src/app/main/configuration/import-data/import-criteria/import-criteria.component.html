Pasos para importar los neumáticos

PASO 1 Solo arrastre el archivo excel de la plantilla realizada

PASO 2 Una vez seleccionado el archivo se te abrira una ventana de los datos importados con sus validaciones correspondientes

PASO 3 Para verificar si los datos se relacionaron con el sistema, haz click en Ir a Criterios de Aceptacion y te rediccionará a la lista de Neumáticos.
